Detailed Explanation about The Project

  • Markdown is a lightweight markup language that is used to format text in a way that can be easily converted to HTML or other document formats. A Markdown viewer project involves creating an application that can render Markdown files and display them in a user-friendly format. This type of project can be useful for anyone who wants to read Markdown files without having to use a text editor or command-line tool.
    • Front-end: Reactjs
    • Back-end: Nodejs
    • Database: MongoDB
  • Secured by login system with JWTwebtoken, also implemented password reset flow using nodemailer
  • Axios is used to fetch the data. Formik & yup, used for forms and validations. Used Bootstrap for designing.
  • Once logged in React markdown is used to render Markdown files and display them in a user-friendly format
  • Users can save the data for future use whenever they logged in again and also they can update their data
  • Project has a help button, when it is clicked it will open a pdf file that contains all the details of the markup language, it is also in downloadable format.
